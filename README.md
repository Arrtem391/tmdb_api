## hello_api_TMDB.py
- Проверяет работу с API TMDB, получая бюджет конкретного фильма по его ID. Используется для тестирования API ключа.
- **Что делает:** Проверяет подключение к API TMDB, получает бюджет фильма по его ID
- **Вводные данные:** API ключ от TMDB
- **Зависимости:** Требует только API ключ, можно запускать первым
## make_own_db.py
- Создает локальную базу данных фильмов, запрашивая информацию с API TMDB. Сохраняет данные в JSON файл для последующего использования.
- **Что делает:** Создает локальную базу данных фильмов из API TMDB
- **Вводные данные:** API ключ от TMDB
- **Зависимости:** Требует API ключ, создает файл MyFilmDB.json для других скриптов
## find_similar.py
- Находит похожие фильмы на основе совпадений по коллекции, языку, бюджету и жанрам. Выдает персонализированные рекомендации.
- **Что делает:** Ищет похожие фильмы по различным параметрам (коллекция, язык, бюджет, жанры)
- **Вводные данные:** Путь к файлу базы данных и название фильма для поиска
- **Зависимости:** Требует созданную базу данных от make_own_db.py
## search_in_db.py
- Осуществляет поиск фильмов по названию в локальной базе данных. Находит все фильмы, содержащие введенное ключевое слово.
- **Что делает:** Осуществляет поиск фильмов по названию в локальной базе данных
- **Вводные данные:** Путь к файлу базы данных и ключевое слово для поиска
- **Зависимости:** Требует созданную базу данных от make_own_db.py
