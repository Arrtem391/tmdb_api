## Скрипт: hello_api_TMDB.py

### Что делает скрипт?
Проверяет корректность работы с API кинобазы данных, получая и отображая бюджет конкретного фильма. Используется для тестирования подключения к сервису.

### Как его запустить?
```bash
python hello_api_TMDB.py
```

### Что он требует для запуска?
- Ключ доступа к API кинобазы данных
- Стандартная установка Python 3.x

### Где взять необходимые для его запуска штуки?
Ключ API можно получить после регистрации на сайте кинобазы данных в разделе настроек разработчика.

### Что он выведет?
После ввода ключа скрипт отобразит бюджет тестового фильма в числовом формате.

**Пример работы:**
```
Введите ключ API: [скрытый ввод]
4000000
```

## Скрипт: make_own_db.py

### Что делает скрипт?

# Документация для скрипта find_similar.py

## Что делает скрипт?
Находит и рекомендует фильмы, похожие на указанный пользователем, используя систему взвешенных критериев для оценки схожести. Скрипт анализирует несколько характеристик фильмов и вычисляет степень их соответствия целевому фильму.

## Как он это делает?
Система рекомендаций работает по принципу начисления баллов за совпадение различных параметров:

### Критерии схожести и их вес:
- **Принадлежность к одной коллекции** - наибольший вес, так как фильмы из одной серии обычно имеют общую стилистику и тематику
- **Совпадение жанров** - высокий вес, поскольку жанр сильно влияет на настроение и содержание фильма
- **Одинаковый язык оригинала** - средний вес, что указывает на культурную и региональную близость
- **Схожий бюджет** - меньший вес, так как бюджет может коррелировать с масштабом производства

Каждому фильму в базе присваивается суммарный балл на основе этих критериев, после чего формируется список рекомендаций в порядке убывания схожести.

## Как его запустить?
```bash
python find_similar.py
```

## Что он требует для запуска?
- Файл с локальной базой данных фильмов (создается скриптом make_own_db.py)
- Точное название фильма, для которого нужно найти похожие

## Где взять необходимые для его запуска штуки?
Базу данных необходимо предварительно создать с помощью скрипта make_own_db.py.

## Что он выведет?
Список из 8 наиболее похожих фильмов, отсортированный по степени схожести.

**Пример работы:**
```
Введите путь к базе данных: MyFilmDB.json
Введите фильм для поиска: Inception
Матрица
Интерстеллар
Престиж
Начало
Исходный код
Помни
Сквозь снег
Ванильное небо
```

### Что означают числа в параметрах?
Это весовые коэффициенты, определяющие важность каждого критерия при расчете схожести фильмов. Более высокие значения означают, что данный параметр сильнее влияет на итоговую рекомендацию.

### Что такое "рейтинг" в понимании этого скрипта?
В контексте этого скрипта "рейтинг" - это не оценка качества фильма, а числовая величина, отражающая степень схожести с целевым фильмом. Чем выше это число, тем больше общих характеристик у фильмов.

### Зачем удаляется целевой фильм из результатов?
Чтобы избежать рекомендации того же самого фильма, для которого ищутся похожие - это было бы бессмысленно для пользователя.



## Скрипт: search_in_db.py

### Что делает скрипт?
Выполняет поиск фильмов по названию в локальной базе данных, находя все киноленты, содержащие указанную фразу.

### Как его запустить?
```bash
python search_in_db.py
```

### Что он требует для запуска?
- Файл с локальной базой данных фильмов
- Ключевое слово для поиска

### Где взять необходимые для его запуска штуки?
Базу данных необходимо предварительно создать с помощью соответствующего скрипта.

### Что он выведет?
Алфавитный список всех фильмов, в названиях которых встречается указанное ключевое слово.

**Пример работы:**
```
Введите путь к базе данных: MyFilmDB.json
Введите фильм для поиска: star
Звезда родилась
Звездные войны
Звезда
...
```
